# Purpose

This document defines the athlete's current profile.

Unlike activity JSON files, which describe individual workouts, this
document describes the athlete's long-term characteristics, goals, and
training environment.

It provides context that allows activity analyses to be interpreted
appropriately.

This document should only be updated when meaningful long-term changes
occur.

Examples include:

- new training goals
- significant fitness improvements
- new equipment
- revised training schedule
- updated threshold pace

It should **not** be updated after every workout.

# Success Indicators

The athlete is progressing when one or more of the following improve
over time.

Longer continuous running duration.

Lower heart rate at the same pace.

Faster pace at the same heart rate.

Reduced HR drift.

Improved workout compliance.

Improved pacing consistency.

Improved cadence stability.

Greater weekly consistency.

---

# Warning Indicators

Potential concerns include:

Persistent HR drift.

Repeated missed workouts.

Increasing recovery requirements.

Declining cadence.

Declining consistency.

Repeated pacing errors.

Multiple consecutive regressions.

These indicators should prompt closer observation rather than immediate
training changes.

# Guiding Principle

The athlete profile provides context, not conclusions.

It exists to ensure that every activity is interpreted relative to the
athlete's current abilities, goals, and training philosophy rather than
against generic performance standards.
